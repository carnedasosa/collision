System Requirements

Technical Rider

Hybrid Set with Piano

  • M
    Pioneer DJ DJM-V10-LF

    Professional mixer (LF version preferred).

  • C
    4 x AlphaTheta CDJ-3000X / Pioneer CDJ-3000

    All units must be updated to the latest firmware version and run the same version.

  • E
    1 x Ethernet Switch (No Wi-Fi)

    With 5 (five) CAT5e or higher cables, connected via Pro DJ Link to the CDJs and mixer.

  • R
    Pioneer DJ RMX-1000

    Connected via <<Multi I/O>> send and return channels (with jack cables on both ends) to the DJM-V10.

  • S
    2 x Stereo Booth Monitors

    Positioned at ear level, powered separately with independent control. Preferred: Meyer Sound, L-Acoustics, d&b.

DJS AND PIANIST MUST BE CLEARLY VISIBLE TO THE AUDIENCE

Piano Configuration

Primary Instrument

Yamaha P-225 Black, connected to Channel 1 of the DJM-V10 in LINE mode. Piano without furniture stand.

Positioned next to the DJ on the left side, allowing space for simultaneous operation.

Hardware Requirements

Audibax Onyx Pro 1 adjustable piano stand (or similar to allow for high positioning). Note: Artist is 1.97m tall.

Sustain Pedal: Weighted Yamaha FC4A, FC3A, or Roland DP-10 (No lightweight plastic switches).

88-Key Alternatives: Nord Stage 4, Nord Stage 3, Kawai MP7SE, Kawai MP11SE.

Visual Setup Explanation

RCA / Digital (CDJs)
Jack TRS IO Send (RMX)
Jack TRS IO Return (RMX)
Jack TS Channel 1 (Piano)

Space

Lighting
